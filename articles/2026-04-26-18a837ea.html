一、卦牌 App：選不混占星塔羅那種

上週四晚上十點半，一個做寵物食品代工的老闆在 LINE 傳訊息給我，問的不是 Claude Code 怎麼裝，是「老師我手邊那本《周易本義》翻爛了，但我每天通勤兩小時，總不能在捷運上一直翻書吧，有沒有什麼 App 可以練?」

我盯著那則訊息看了一會兒。因為這問題我這兩年被問了至少四十次。

先講結論：能每天練的工具有，但好用的、不會讓你練成「占卜娛樂思維」的，少。我自己長期在用、也會推薦給學員的，大概就三種搭配。

一、卦牌 App：選不混占星塔羅那種

市面上「易經 App」搜出來大概有上百個，但九成是商業占卜應用——配色花俏、首頁推「今日運勢」「事業運桃花運」，那種我直接跳過。對中小企業主來說，這類 App 反而會讓你滑進「等老天給答案」的被動心態，跟我教易經想培養的決策視角剛好相反。

我比較常推學員用的是兩個方向。一個是「卦象查詢工具」類，像台灣的「易經卜卦」（介面很陽春那種），純粹當電子版索引，搖出卦來自己看卦辭、爻辭、彖傳。優點是不會幫你翻譯成「你最近會遇到貴人」這種廢話，缺點是介面真的醜，上次我學員看完跟我說「老師這 App 看起來像 2008 年做的」——對，但堪用。

另一個方向是用 Notion 或 Obsidian 自己做卦牌資料庫。我自己用 Obsidian 寫了 64 個卦的筆記檔，每個檔裡放卦辭、自己的解讀、過去遇過印證的真實案例。每天通勤滑一個卦看一遍，三個月過完一輪 64 卦，比 App 推播有用太多。

二、把練卦變成日常的關鍵：不是工具，是場景

說實話，工具只是 30%。剩下 70% 是你怎麼把練卦這件事「黏」進生活。

我自己的方法很笨：每天早上煮咖啡的那五分鐘，隨機抽一個卦，然後問自己「這個卦如果落在我今天要做的決定上，它會怎麼提醒我？」就這樣。不用搖銅板、不用焚香、不用儀式感。

上個月一個做精密機械零件的客戶跟我學這套，他原本以為要花很多時間。結果兩週後跟我說，他現在開晨會前會抽一卦看五分鐘，反而比以前直接衝會議室更穩。他說最有感的一次是抽到「需」卦——卦辭講「有孚，光亨，貞吉，利涉大川」——那天他本來要在會議上拍板一個 800 萬的設備投資案，結果看完卦辭裡那句「需」字，硬是把決策延後一週去查供應商的財報。後來發現那家供應商三個月前才剛被法院假扣押。

他跟我說這話的時候，眼神是真的有點發亮。

三、把 AI 拉進來：Claude Code + 卦牌的奇怪組合

這是我最近才在試的玩法，還在摸索階段，不確定會不會普及。

我把 64 卦的卦辭、爻辭、自己累積的案例筆記全部丟進 Claude 的 context 裡，做了一個「易經決策助理」。當我遇到一個具體的經營問題——例如「該不該開分公司」——我先自己抽一卦，然後把問題跟卦辭一起餵給 Claude，讓它從那個卦的角度幫我拆「卦辭裡哪幾個字跟我這個問題相關、為什麼相關、可能在提醒什麼風險」。

這不是占卜。這比較像是用 AI 當一個「易經視角的陪練」。它不會給我答案——卦辭本來就不給答案——但它會幫我把我自己想得不夠清楚的地方用另一個語言再講一遍。

企業 AI 導入這件事，很多老闆以為是要把所有判斷交給機器。我看到的反而是反過來：機器幫你算清楚理性那一面，模糊的、跟人有關的那一面，留給東方思維。Claude Code 教學我講了快兩年，但中小企業數位轉型真正卡住的點，從來不是工具——是老闆面對「該不該」的時候缺一個第二視角。

一個小提醒

如果你是剛開始想練卦牌的人，我老實說：別買貴的 App，別下載一堆。先把《易經》前八卦（乾、坤、屯、蒙、需、訟、師、比）這八個搞熟，比你下載十個 App 都有用。卦牌不是工具的問題，是熟不熟的問題。

我自己看卦看了二十年，前年還是在「漸」卦的「鴻漸于陸，夫征不復」那段被敲了一下——那時候我正在猶豫要不要把曝光引擎這個案子推掉。後來沒推。

工具會換，App 會倒，但卦辭那幾千年累積下來的人性觀察不會。

常見問題

我完全沒讀過易經，直接下載 App 來練會不會根本看不懂？
會。而且很多人就是這樣放棄的。我建議先花兩三週把乾、坤兩卦的卦辭、彖傳、象傳搞熟（只有兩卦，網路上免費資料一堆），有了基本語感再開 App。直接從 App 入門最大的陷阱是它會給你「白話翻譯」，那個翻譯通常很爛，會讓你以為自己懂了，其實是被 App 編輯誤導了一輪。
用 AI 來輔助看卦會不會褻瀆易經？
我以前也覺得會。後來想通了——古人用銅錢、用蓍草，那個年代那就是他們的「工具」。AI 只是新一代的工具。重點不是用什麼搖卦，是你有沒有真的把卦辭當回事去想自己的問題。我用 Claude 拆卦辭的時候，反而比以前自己悶頭看更願意承認「這句話我其實沒看懂」。
每天練卦真的會對做生意的決策有幫助嗎，還是只是心理安慰？
老實講，初期確實有心理安慰的成分，這沒什麼好否認。但練到第三個月之後會出現一個轉變——你開始能在問題還沒成形之前就感覺到「這個局不對勁」。那不是玄學，是六十四卦看了一輪又一輪之後，你的腦袋被訓練出一種「結構辨識力」。我有個做餐飲連鎖的學員練了八個月，他說現在看新店面選址，腦袋會自動冒出某幾個卦的畫面。這個我不知道怎麼解釋，但我自己也有過。
