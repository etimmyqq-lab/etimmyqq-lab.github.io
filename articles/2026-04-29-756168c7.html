人才任用的「最終判斷」，究竟是什麼

易經卦牌可以用來決定要不要雇用一個人嗎？我的答案是：可以，但不是你想的那種用法

上個月陪一家做工業零件的老闆開了三次會，到第三次他終於說出他真正卡住的地方：「我明明在 104 上面看到一個背景完整的業務主管，面試表現也不錯，但我就是感覺哪裡不對，又說不上來。我那天抽了一個卦，得了『履卦』，卦辭說『履虎尾，不咥人，亨』——你說這是什麼意思？」

我沒有直接回答他。

因為他問的其實不是易經，他問的是：我這個感覺算不算數？

人才任用的「最終判斷」，究竟是什麼

先說一個我自己踩過的坑。

幾年前協助一家科技電子廠做組織診斷，他們的 HR 主管做事很細，程序都走完了：背景調查、三輪面試、技能測驗。最後進來的那個人各項分數都高，但入職六個月後變成部門最大的麻煩製造者。我後來跟那個 HR 主管聊，她說：「那個人面試時太完美了，我就有一點感覺不對，但沒有證據，就算了。」

有沒有工具可以量化「直覺感覺不對」這件事？

現在的 AI 工具可以做另一層：用 Claude Code 分析求職者的工作經歷，幫你自動整理跟職位的落差；透過問卷資料做跨候選人比較；甚至把過去成功員工的特質輸入進去，讓模型幫你找相似性。這些我都在幫客戶建。

但「這個人放進我的公司、我的文化、現在這個時間點，對不對？」這個問題，資料是回答不了的。

易經卦牌在人才任用裡的實際角色

我說「可以用」，但要說清楚用在哪裡。

易經不是人力資源測評系統。它不是 DISC 測驗，不會給你一個人的個性分類。如果有人說「這個人抽到某卦，所以適合做哪種工作」，我覺得這跑偏了。

易經卦牌真正有價值的地方，是在你已經收集完資訊之後，幫你把內心真正在擔心什麼逼出來。

那個老闆抽到的「履卦」，卦辭裡有一個意象——踩在老虎尾巴上，沒被咬，還走過去了。我問他：「你覺得你現在踩的是什麼？是這個人的哪個特質讓你感覺危險？」

他想了一下說：「他之前的公司跟我們的競爭對手有合作關係。」

那才是他真正的擔心。不是什麼感覺不對，是有一個具體的風險點他沒有說出口。

卦牌做的事，是在資料充分但決定還沒落地的那個空白裡，逼你說話。不是替你做決定，是讓你聽清楚自己的聲音。

「最終判斷」這個說法本身就有問題

回到原來的問題：易經卦牌可以用來做人才任用的「最終判斷」嗎？

如果「最終判斷」的意思是：不管前面做了什麼，就用這個卦來決定錄不錄用——我的答案是不建議。

這樣用會出兩個問題。第一，你把責任外包給卦象，等於放棄了自己作為決策者的判斷力。第二，如果出了事，你的解釋是「我問過易經」，這解釋不了任何事。

但如果「最終判斷」的意思是：在所有流程都走完、資料都看了、直覺還是在動盪的那個當下，用一個外部視角把自己拉出來，重新問一次「我到底在擔心什麼」——這我覺得是用對地方了。

我自己在做企業 AI 導入顧問的過程裡，也會在某些關鍵時刻用卦。不是問「這個客戶能不能做」，而是當我對某個方案感覺卡住的時候，用卦辭逼自己把那個卡的點說清楚。結果常常是：卦辭本身沒那麼重要，但我為了解釋那個卦，反而說出了我真正的顧慮。

中小企業數位轉型和人才佈局的過程裡，這種「說清楚自己在顧慮什麼」的能力，其實比任何工具都重要。

好的卦讀，是讓你聽清楚自己；爛的卦讀，是替你做選擇。

---

那個老闆最後有沒有雇用那個業務主管？

他後來去查了，那個合作關係確實存在，但已經結束兩年多了。他評估了一下，決定錄用，附帶條件是簽了一份競業限制條款。

我問他：「那個卦最後幫到你了嗎？」

他說：「不是卦幫我，是我終於把那個擔心說出來了。」

這句話比我能解釋的任何東西都準確。

常見問題

易經卦牌跟 MBTI、DISC 這些測驗有什麼不同？可以配合用嗎？
可以配合，但用途完全不同。MBTI 和 DISC 是幫你描述一個人的特質分布，是向外看的工具——你在評估那個人。易經卦牌是向內看的，它逼你檢視自己身為決策者的狀態和盲點。兩個一起用的話，我通常是先跑完測評資料，在需要拍板前用卦牌做一次自我對話。不是替代，是補缺。
如果我不懂易經，可以請顧問幫我解卦來做人才決策嗎？
可以請人幫解讀卦象，但最後的決定不應該讓顧問替你下。我幫客戶解卦的方式是把卦辭的幾個意象拿出來，問他：「哪一個跟你現在的狀況有共鳴？」答案是他自己說的，不是我告訴他的。如果有人幫你解卦之後直接說「這個人可以用」或「不能用」，建議換一個顧問。
用 Claude Code 做候選人分析，跟用易經卦牌，這兩件事可以同時做嗎？
不只可以，我覺得這才是比較完整的做法。Claude Code 幫你把客觀資料整理清楚、把盲點照出來；易經卦牌幫你在面對那堆資料之後、還沒決定的那個時刻，把自己的真實顧慮逼出來。一個往外看，一個往內看。多數人才決策失敗不是因為資料不夠，而是因為決策者沒有說出自己真正在擔心什麼。
