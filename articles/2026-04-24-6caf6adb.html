我一開始也不信

上週三下午，一位做水五金外銷的老闆娘傳訊息給我，劈頭就問：「Timmy老師，你用易經幫客戶決策，會不會被笑啊？」我笑了一下，回她：「會。上個月才被笑過。」

這個問題我被問過不下三十次。今天不打算繞圈子，直接講我自己的立場——易經沒有「預測未來」的科學根據，但它作為決策輔助工具，有相當紮實的認知科學基礎。 這兩件事不一樣，混在一起就是玄學；拆開看，它就是一套被用了三千年的思考框架。

我一開始也不信

先講個誠實的。我會碰易經，是因為 2019 年家裡有人過世，當時朋友送了一本《周易本義》，我隨手翻，翻到「履虎尾，不咥人」那句，愣住。不是覺得神準，是覺得——怎麼有人三千年前就把那種「明知道有風險但得硬著頭皮走過去」的狀態，寫得這麼精確。

後來我才知道，榮格（Carl Jung）晚年寫《易經》德文版序時，用了一個詞叫「共時性原理」（Synchronicity）。他不是說易經能預測未來，他是說：當人把一個混亂的情境拋給一個隨機符號系統，然後逼自己用這個符號去重新詮釋情境，這個「被逼著換角度看」的過程本身，就會產生新的認知。

這跟企業 AI 導入裡的一個老問題其實很像——老闆卡住的時候，通常不是資料不夠，是他看事情的框架卡住了。

回到那個「有沒有科學根據」的問題

分兩層講。

第一層，如果你問的是「卦象能不能物理上預測未來」——沒有。這點我不打算幫它辯護。你丟三枚銅板六次，得到的卦，跟宇宙的因果律沒有任何可驗證的關聯。這部分是玄學，我不賣這個。

第二層，如果你問的是「為什麼很多做決策的人用了之後覺得有用」——這有認知科學的解釋。

心理學上有個概念叫「決策疲勞」（decision fatigue）。中小企業主一天要做四五十個決定，從「這批貨報關要不要加急」到「老陳的薪水要不要調」，到最後他對任何一個決策都提不起細想的力氣。這時候，一個隨機產生的「外部視角」——不管是易經、塔羅、還是擲硬幣——它的功能不是給答案，是強迫你停下來，換個角度再想一次。

我去年陪一家做塑膠射出的客戶，老闆要不要接一筆陸商的大單，他糾結兩週。我們用 Claude Code 幫他跑了財務模型、算了產能負荷、分析了客戶信用。數字全部支持他接。他還是不敢。

最後我們抽了一個卦，是「訟」。卦辭講的是爭訟、糾紛、就算你贏也贏得不痛快。他看完沉默了十秒，說：「其實我一直覺得這家客戶有問題，但說不出來。」三個月後那家陸商真的出事，他沒接那單，閃掉一次災。

卦靈嗎？我不知道。但易經提供的那個「外部框架」，幫他把自己壓抑的直覺講出來了。這件事有科學根據——心理學家 Gary Klein 的「自然決策理論」（Naturalistic Decision Making）講的就是這個：專家的直覺通常是對的，但人會用理性去壓制它，直到某個外部刺激把它釋放出來。

所以我怎麼用它

在企業 AI 導入的案子裡，我把易經放在很後面的一個環節。前面九成的工作，是理性的——流程拆解、成本分析、Claude Code 幫老闆把 SOP 數位化、跑報表、寫客服腳本。這些是技術能解的問題。

到最後那一步，老闆臉色變了、開始抽菸、太太打電話來關心——這個時候，卦才上桌。

因為這個時候的問題不是「數據支不支持」，而是「我心裡那塊還沒鬆開」。易經的功能是幫他把那塊看清楚。

有時候抽出來的卦很直白，像「乾」——就是往前推；有時候很曖昧，像「屯」——起步艱難。我不會告訴他「卦說你要做 X」，我會問他：「看到這個卦，你第一個想到的是什麼？」然後那個真正卡住他的東西就會出來。

這不是預測，是諮商。

寫在最後

如果有人跟你說易經能算明牌、能算股票明天漲跌，那是騙子。但如果有人跟你說易經是一套強迫你多維度思考的結構化工具，在決策疲勞的狀態下重新激活直覺，這個說法，我覺得站得住腳。

中小企業數位轉型最難的從來不是技術那一段。技術那段 Claude Code 就能解掉一大半。最難的是老闆願不願意踏那一步。而那一步踏不踏得出去，往往不是理性能回答的。

常見問題

那我是不是要先學會易經才能用？
不用。我自己也不是背熟六十四卦那種。大部分時候我就是在卦辭旁邊放現代白話翻譯，看到哪句打中你，就停在那句想一下。重點是「換個框架重看」這件事，不是把你變成國學大師。
這跟去廟裡擲筊有什麼不一樣？
擲筊是問神、求答案；易經是給你一個符號，逼你自己把答案講出來。前者是把決策權交出去，後者是把決策權逼回你自己手上。差很多。
如果老闆就是不信這套，還能用嗎？
當然。我不會強推。有些老闆就是鐵齒理性派，我們就全程用資料、模型、Claude Code 跑到底，也一樣能做出好決策。易經只是我工具箱裡的一個選項，不是必備。有人用得上，有人用不上，這很正常。
